# Personal Finance Tracker (Java)

**Domain:** Finance & Banking

## Features
- Add Income/Expense
- View all transactions
- Category-wise pie chart
- Data saved in SQLite

## How to Run
1. Download repository
2. Open CMD in folder
3. Run:
javac -cp "lib/" Src/.java
java -cp ".;lib/*" Main
